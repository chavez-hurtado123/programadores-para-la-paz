Comando uname -a: Muestra información detallada del kernel del sistema operativo, arquitectura y versión.
Comando whoami: Muestra el nombre del usuario actual con el que se ha iniciado sesión en la terminal.

Es vital conocer esta información al administrar un servidor para saber qué permisos tenemos, qué versión del sistema estamos usando para instalar programas compatibles y para identificar posibles problemas de compatibilidad o seguridad.

El login es el proceso mediante el cual una aplicación valida las credenciales de un usuario para verificar su identidad. Es fundamental porque permite restringir el acceso a funciones críticas o datos sensibles, asegurando que solo usuarios autorizados realicen acciones como crear causas o gestionar reportes. Algunas rutas deben estar protegidas, como las de administración o perfil privado, para evitar intrusiones o modificaciones no autorizadas por terceros. Bajo ninguna circunstancia se deben exponer públicamente contraseñas, documentos de identidad, números telefónicos, tokens de autenticación o claves de API. Proteger esta información es un requisito ético y técnico para garantizar la confianza y seguridad de la comunidad dentro de la plataforma.
